# Meteo Data Analysis

A Python data science project that analyzes 30 years (1990-2020) of historical weather data, focusing on temperature trends and distributions. 

**Cloud-Based Processing:** To handle the massive 7.7 GB dataset efficiently, this project is designed to run in a cloud environment (like Google Colab). It uses the `kagglehub` library to fetch the dataset directly into the cloud server's memory, completely bypassing the need for local storage or heavy local computing power.

**Libraries used:** `pandas`, `xarray`, `matplotlib`, `kagglehub`.
